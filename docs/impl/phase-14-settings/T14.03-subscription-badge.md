## Implementation
```typescript
const { data: subscription } = useSubscription()

const getPlanBadge = () => {
  if (subscription?.status === 'trialing') {
    return <Badge label="Trial" variant="info" />
  }
  if (subscription?.plan === 'pro') {
    return <Badge label="Pro" variant="success" />
  }
  if (subscription?.plan === 'premium') {
    return <Badge label="Premium" variant="success" />
  }
  return <Badge label="Free" variant="neutral" />
}

<ListRow
  leftIcon={<CrownIcon color="#FFC107" />}
  title="Subscription"
  rightElement={getPlanBadge()}
  showChevron
  onPress={() => navigation.navigate('Subscription')}
/>
```

## Visual Specs
- Badge: colored based on plan
- Free: neutral gray
- Trial: info blue
- Pro/Premium: success green
