## Implementation
```typescript
<View style={styles.section}>
  <Text style={styles.sectionTitle}>Do Not Disturb</Text>
  <ToggleRow
    label="Quiet Hours"
    description="Pause notifications during set hours"
    value={settings.quietHoursEnabled}
    onValueChange={(v) => updateSetting('quietHoursEnabled', v)}
  />
  {settings.quietHoursEnabled && (
    <View style={styles.timeRange}>
      <TimePicker
        label="From"
        value={settings.quietHoursStart}
        onChange={(t) => updateSetting('quietHoursStart', t)}
      />
      <Text style={styles.to}>to</Text>
      <TimePicker
        label="To"
        value={settings.quietHoursEnd}
        onChange={(t) => updateSetting('quietHoursEnd', t)}
      />
    </View>
  )}
</View>
```

## Visual Specs
- Toggle: enables/disables quiet hours
- Time range: shown only when enabled
- From/To: side by side with "to" between
