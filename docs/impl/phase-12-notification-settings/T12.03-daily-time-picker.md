## Implementation
```typescript
const [showTimePicker, setShowTimePicker] = useState(false)

<Pressable onPress={() => setShowTimePicker(true)} style={styles.timePicker}>
  <Text style={styles.timeLabel}>Reminder Time</Text>
  <View style={styles.timeValue}>
    <Text style={styles.timeText}>{formatTime(settings.reminderTime)}</Text>
    <ChevronDownIcon size={20} color="#9E9E9E" />
  </View>
</Pressable>

<DateTimePicker
  visible={showTimePicker}
  mode="time"
  value={settings.reminderTime}
  onChange={(time) => {
    updateSetting('reminderTime', time)
    setShowTimePicker(false)
  }}
/>
```

## Visual Specs
- Row: label on left, time + chevron on right
